# WME Show Editor Names

A Tampermonkey script for the Waze Map Editor that displays editor names on the map.

## Features

- Shows editor names who are active in the screen
- Integrates seamlessly with the Waze Map Editor interface
- Lightweight and performance-optimized

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) browser extension
2. Click [here](https://update.greasyfork.org/scripts/562181/WME%20Show%20Editor%20Names.user.js) to install the script
3. Confirm the installation when prompted
4. Reload the Waze Map Editor page

## Usage

Once installed, the script automatically runs in the Waze Map Editor. Editor names will be displayed based on the current map view and zoom level.

## Requirements

- Waze Map Editor account
- Tampermonkey extension
- Modern browser (Chrome, Firefox, Safari, Edge)
- Turn on active editor
